# WAIR — Who Am I Really

[![Ko-fi](https://ko-fi.com/img/githubbutton_sm.svg)](https://ko-fi.com/asharpdude)

A modern replacement for Microsoft BGInfo. WAIR lives in your system tray and gives you instant access to key system information with one-click copy to clipboard.

**Developed by Adam Sharp of [E-Hounds, Inc.](https://www.ehounds.com)**

---

## Download

Head to the [**Releases**](https://github.com/ehounds/WAIR-releases/releases) page and download the latest `WAIR Setup x.x.x.exe` installer.

### No prerequisites required
The installer bundles everything needed. No Node.js, no .NET Framework, no wmic — just download, install, and run on any Windows 10/11 system.

---

## Features

- **System tray app** — unobtrusive, always available, click to open
- **Large, readable type** — monospace values designed to be read at a glance
- **One-click copy** — hover any field to reveal a copy button
- **Fully configurable** — toggle any field on or off via the settings panel
- **Smart network detection** — automatically identifies the adapter with an active internet gateway
- **Multi-adapter support** — select which NIC's info to display
- **Persistent config** — your selections are saved across restarts

## System Information Fields

| Category | Fields |
|----------|--------|
| **Identity** | Computer Name, FQDN, Logged-in User, Full User (whoami), Domain/Workgroup |
| **OS** | Windows Edition, Windows Version (e.g. 25H2), Build Number, BIOS Version |
| **Hardware** | CPU Model, Total RAM, Serial Number |
| **Network** | IPv4 Address, MAC Address, Default Gateway, DNS Servers, Public IP |
| **Status** | Uptime, Last Boot Time |
| **Storage** | Per-disk breakdown with volume name, capacity, usage %, and free space |
